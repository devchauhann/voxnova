# VoxNova

VoxNova is an advanced Editorial AI Speech Studio designed for creators, podcasters, and content professionals to generate, edit, and manage high-quality voice content with ease. Powered by state-of-the-art AI, VoxNova offers seamless text-to-speech, waveform editing, and customizable voice settings in a modern, intuitive interface.

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/devchauhann/voxnova.git
   cd voxnova
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. **Open your browser:**
   Visit `http://localhost:5173` to view the app.
